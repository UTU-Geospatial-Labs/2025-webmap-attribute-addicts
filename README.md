# After school activities in Turku

**This webmap was created as MAAN7771 course work in 2025 by:**

**Henna Alahäme, Janette Hagren, Kaisa Takala and Martin Zábranský.**

The map visualises after-school activities for kids and youth in Turku, providing routing from schools to nearby activities as an interactive webmap. The solution responded to challenge provided by Lounaistieto.

Explore the map: https://utu-geospatial-labs.github.io/2025-webmap-attribute-addicts/
